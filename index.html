NeuroAdapt Research Build v1.2

Performance Drift
Assessment System

This system measures how performance changes across three standardised cognitive task conditions. All outputs are expressed as metric deltas relative to an individual baseline.

  • Phase 1 — Baseline Reaction Time (30 trials)
  • Phase 2 — Sustained Attention / CPT (60 trials)
  • Phase 3 — Conflict Task / Flanker (80 trials)
  • Optional — Pressure condition RT (30 trials)

Keyboard only. Desktop or laptop required. Approximately 20 minutes.

Research context only. Pre-validation prototype. Output consists of raw trial data and computed metrics only. No clinical interpretation is generated. Results must not be used for clinical, diagnostic, or occupational decisions.

    Methodological Limitations
    · Pre-validation prototype — thresholds not empirically derived.
    · Browser timing (performance.now) has ~5–20 ms jitter, not corrected.
    · No test-retest reliability data exists for this implementation.
    · No normative comparison is computed or implied.
    · Single-session data does not support temporal consistency conclusions.
    · CPT RT logged for correct target hits only (v1.2). False alarm RT not recorded. RT variability reflects hit responses ≥150ms. Drift computable from trial_number in raw JSON.
    · RT baseline/pressure metrics exclude responses <150ms (anticipatory) and >1500ms (lapse). Excluded trials retained in raw JSON.
    · Sound cue depends on device/browser — AudioContext may be silently blocked.
